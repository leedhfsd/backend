# CRUD MARKDOWN-BLOG  
Express, MongoDB를 이용하여 CRUD가 가능한 블로그를 만드는 프로젝트입니다.

# 프로젝트 계획 이유
NoSQL 데이터베이스인 MongoDB와 Express를 이용한 백엔드 개발 프로젝트로 웹 개발의 전체적인 흐름에 대해서 학습하기 위함입니다.

# 설치

```
git clone https://github.com/leedhfsd/markdown-blog.git
```
패키지 설치
```
npm i
```

# 실행

```
npm run devStart
```
